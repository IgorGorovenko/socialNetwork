# Social network

Социальная сеть

## Функции

- Регистрация и вход
- Профиль пользователя
- Пользователи
- Чат
- Комментарии

## Getting Started

#### 1. Clone this repository:

Скопировать проект

```
git clone git@github.com:igor-gorovenko/social_network.git
```

Переходим в скопированный проект, в корневую папку

```
cd social_network
```

#### 2. Install dependencies:

```
npm install
```

## Запуск

```
npm start
```
